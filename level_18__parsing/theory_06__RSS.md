`BeautifulSoup` мощный инструмент для извлечения информации с сайта. 

Но содержит весьма существенные минусы:
* Защита от парсинга, которая может со временем меняться
* Зависимость от вёрстки страницы, которая тоже подвержена изменениям

И в том, и в другом случае проблема требует "ручного" решения.

Но есть и другой способ, напрочь лишённый этих двух недостатков - **RSS**.

RSS расшифровывается как **Really Simple Syndication** — буквально «очень простая синдикация».

* Это **формат XML**, который позволяет сайтам **публиковать обновления** (новости, статьи, блоги) в стандартизированном виде.
* Клиенты (RSS-ридеры, боты, приложения) могут **подписываться на RSS-ленту** и автоматически получать новые записи без захода на сайт.
* Иногда его называют **Rich Site Summary**, но современное и чаще используемое значение — **Really Simple Syndication**.

## Теория: как устроен RSS

RSS — это **XML-документ** со строгой структурой. В нём есть:

* `<channel>` — информация о канале (название сайта, описание, ссылка)
* `<item>` — отдельные публикации (заголовок, ссылка, описание, дата и т.д.)

### Пример структуры RSS 2.0

```xml
<rss version="2.0">
  <channel>
    <title>Название сайта</title>
    <link>https://example.com</link>
    <description>Описание сайта</description>

    <item>
      <title>Заголовок статьи</title>
      <link>https://example.com/article1</link>
      <description>Краткое описание</description>
      <pubDate>Mon, 23 Feb 2026 10:00:00 GMT</pubDate>
    </item>

  </channel>
</rss>
```

---

### Чем RSS отличается от обычного сайта?

| Обычный сайт                    | RSS                               |
| ------------------------------- | --------------------------------- |
| HTML для людей                  | XML для программ                  |
| Нужно заходить вручную          | Обновления приходят автоматически |
| Много лишнего (дизайн, реклама) | Только структурированные данные   |

---

### Где используется RSS?

* Новостные сайты (например, BBC News)
* Блоги (например, Habr)
* Подкасты (например, The Joe Rogan Experience)

#### 4. Парсинг RSS с помощью пакета `feedparser`

```bash
pip install feedparser
```

```python
import feedparser

feed = feedparser.parse("https://example.com/rss")
for entry in feed.entries:
    print(entry.title, entry.link)
```

### Где найти RSS сайта?

Часто:

* `/rss`
* `/feed`
* `/atom.xml`

Или в HTML страницы:

```html
<link rel="alternate" type="application/rss+xml" href="https://example.com/rss">
```
